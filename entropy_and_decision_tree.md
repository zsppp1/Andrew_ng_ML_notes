# 熵函数与决策树

## 1. 熵的定义

信息熵衡量数据的**不确定性/混乱程度**：

```
H(D) = -Σ p(x) · log₂ p(x)
```

| 熵值 | 含义 |
|------|------|
| 高 | 数据混乱、不确定性大 |
| 低 | 数据纯净、分类明确 |
| 0 | 所有样本属于同一类（完全纯净） |

**例子：**
- 50%类别A, 50%类别B → 熵=1（最混乱）
- 100%类别A → 熵=0（完全纯净）

---

## 2. 信息增益

**信息增益 = 分裂前熵 - 分裂后熵**

```
IG(D, A) = H(D) - Σ (|Dv|/|D|) · H(Dv)
```

- 信息增益越大 → 该特征对分类贡献越大
- 决策树每次选择**信息增益最大的特征**进行分裂

---

## 3. 决策树构建过程

```
根节点 [熵高，混杂]
    ↓ 选择最佳特征分裂（信息增益最大）
子节点 [熵降低]
    ↓ 继续分裂
叶节点 [熵≈0，纯净] → 输出分类结果
```

**核心思想：通过分裂不断降低熵，让数据越来越纯净**

---

## 4. 不同算法的分裂标准

| 算法 | 标准 | 特点 |
|------|------|------|
| ID3 | 信息增益 | 偏向多值特征 |
| C4.5 | 信息增益率 | 修正ID3偏向 |
| CART | 基尼系数 | 计算更快 |

**基尼系数：**
```
Gini(D) = 1 - Σ p(x)²
```
与熵本质相似，都是衡量不纯度。

---

## 5. 关键公式速记

| 公式 | 用途 |
|------|------|
| `H(D) = -Σ p·log₂ p` | 计算熵 |
| `IG = H(父) - Σ权重·H(子)` | 计算信息增益 |
| `Gini = 1 - Σ p²` | 基尼系数（CART） |

---

## 6. 一句话总结

> 熵衡量不纯度，决策树通过选择信息增益最大的特征分裂，逐步降低熵，最终到达纯净的叶节点完成分类。
