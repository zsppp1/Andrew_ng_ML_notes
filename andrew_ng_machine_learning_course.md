## 课程一：监督学习 (Supervised Learning)

### Week 1：机器学习概述与单变量线性回归

#### 1.1 机器学习简介
- **定义**：让计算机从数据中学习规律，无需显式编程
- **应用场景**：搜索引擎、推荐系统、语音识别、图像识别

#### 1.2 学习类型分类

| 类型 | 描述 | 示例 |
|------|------|------|
| **监督学习** | 有标签数据，学习输入→输出映射 | 回归、分类 |
| **无监督学习** | 无标签数据，发现数据内在结构 | 聚类、降维 |
| **强化学习** | 通过奖励信号学习最优策略 | 游戏AI、机器人 |

#### 1.3 线性回归模型

**假设函数**：
$$h_\theta(x) = \theta_0 + \theta_1 x$$

**代价函数 (Cost Function)**：
$$J(\theta) = \frac{1}{2m}\sum_{i=1}^{m}(h_\theta(x^{(i)}) - y^{(i)})^2$$

**目标**：找到使 J(θ) 最小的参数 θ₀, θ₁

#### 1.4 梯度下降算法

**更新规则**：
$$\theta_j := \theta_j - \alpha \frac{\partial}{\partial \theta_j} J(\theta)$$

其中 α 为学习率 (learning rate)

**学习率选择**：
- 太小 → 收敛速度慢
- 太大 → 可能不收敛或发散
- 建议：尝试 0.001, 0.01, 0.1, 1...

---

### Week 2：多变量线性回归

#### 2.1 多特征表示

**假设函数**：
$$h_\theta(x) = \theta_0 + \theta_1 x_1 + \theta_2 x_2 + ... + \theta_n x_n$$

**向量化形式**：
$$h_\theta(x) = \theta^T x$$

#### 2.2 特征缩放 (Feature Scaling)

**目的**：让不同特征在相近范围内，加速梯度下降收敛

**方法**：
- **归一化**：$x' = \frac{x - \min}{\max - \min}$
- **标准化**：$x' = \frac{x - \mu}{\sigma}$

**目标范围**：-1 到 1（或相近范围）

#### 2.3 特征工程

- 从现有特征创建新特征
- 例如：$x_3 = x_1 \times x_2$ 或 $x_4 = x_1^2$
- 可将线性模型转化为非线性

#### 2.4 多项式回归

通过特征工程，线性回归可拟合非线性关系：
$$h_\theta(x) = \theta_0 + \theta_1 x + \theta_2 x^2 + \theta_3 x^3$$

---

### Week 3：逻辑回归

#### 3.1 二分类问题

**Sigmoid 函数**：
$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

将任意值映射到 (0, 1) 区间

**假设函数**：
$$h_\theta(x) = \sigma(\theta^T x)$$

输出概率：P(y=1 | x)

#### 3.2 决策边界

- 若 h(x) ≥ 0.5 → 预测 y=1
- 若 h(x) < 0.5 → 预测 y=0
- 决策边界：θᵀx = 0

#### 3.3 代价函数

**对数损失 (Log Loss)**：
$$J(\theta) = -\frac{1}{m}\sum_{i=1}^{m}[y^{(i)}\log(h_\theta(x^{(i)})) + (1-y^{(i)})\log(1-h_\theta(x^{(i)}))]$$

**优点**：凸函数，保证收敛到全局最优

#### 3.4 正则化

**目的**：防止过拟合

**L2 正则化**：
$$J(\theta) = J_{base}(\theta) + \frac{\lambda}{2m}\sum_{j=1}^{n}\theta_j^2$$

| λ值 | 效果 |
|-----|------|
| λ=0 | 无正则化，可能过拟合 |
| λ过大 | 高度正则化，可能欠拟合 |
| λ适中 | 平衡，最佳拟合 |

### Week 4：神经网络基础

#### 4.1 神经网络结构

```
输入层 → 隐藏层 → 输出层
  x   →   a   →   y
```

#### 4.2 神经元模型

每个神经元执行：
$$a = \sigma(Wx + b)$$

- W：权重矩阵
- b：偏置
- σ：激活函数

#### 4.3 前向传播

逐层计算激活值：
$$a^{[1]} = \sigma(W^{[1]}x + b^{[1]})$$
$$a^{[2]} = \sigma(W^{[2]}a^{[1]} + b^{[2]})$$
$$\hat{y} = a^{[L]}$$

#### 4.4 常用激活函数

| 函数 | 公式 | 特点 |
|------|------|------|
| **Sigmoid** | $\frac{1}{1+e^{-z}}$ | 输出 (0,1)，二分类 |
| **ReLU** | $\max(0, z)$ | 计算快，稀疏激活 |
| **Tanh** | $\frac{e^z-e^{-z}}{e^z+e^{-z}}$ | 输出 (-1,1) |
| **Softmax** | $\frac{e^{z_i}}{\sum e^{z_j}}$ | 多分类，输出概率分布 |

## 课程二：高级学习算法 (Advanced Learning Algorithms)

### Week 1：神经网络深入

#### 1.1 Softmax 回归

**用途**：多分类问题

**输出**：
$$a_i = \frac{e^{z_i}}{\sum_{j=1}^{K} e^{z_j}}$$

所有输出之和 = 1，表示各类别概率

#### 1.2 损失函数

**交叉熵损失**：
$$L = -\sum_{j} y_j \log a_j$$

---

### Week 2：神经网络训练

#### 2.1 数据集划分

| 数据集 | 用途 | 比例建议 |
|--------|------|----------|
| 训练集 | 训练模型参数 | 60-80% |
| 验证集 | 调整超参数、选择模型 | 10-20% |
| 测试集 | 最终评估模型性能 | 10-20% |

#### 2.2 偏差与方差

| 状态 | 特征 | 解决方法 |
|------|------|----------|
| **高偏差 (欠拟合)** | 训练误差高 | 增加模型复杂度、更多特征 |
| **高方差 (过拟合)** | 训练误差低，验证误差高 | 更多数据、正则化、简化模型 |
| **低偏差低方差** | 训练和验证误差都低 | 理想状态 |

#### 2.3 正则化技术

| 技术 | 描述 |
|------|------|
| **L2正则化** | 惩罚大权重 |
| **Dropout** | 训练时随机丢弃神经元 |
| **数据增强** | 增加训练数据多样性 |
| **早停** | 验证误差不再下降时停止训练 |

---

### Week 3：模型评估与改进

#### 3.1 学习曲线

横轴：训练样本数量
纵轴：误差

**分析**：
- 训练误差随数据增加而上升
- 验证误差随数据增加而下降
- 数据足够多时，两者趋于接近

#### 3.2 迁移学习

**步骤**：
1. 在大数据集上预训练模型
2. 在目标任务上微调

**适用场景**：目标任务数据量较少

### Week 4：决策树

#### 4.1 决策树结构

```
        根节点
       /      \
    条件A    条件B
    /  \      /  \
  叶节点  叶节点 ...
```

#### 4.2 分裂标准

**信息熵**：
$$H(D) = -\sum p(x) \log_2 p(x)$$

**信息增益**：
$$IG(D, A) = H(D) - \sum \frac{|D_v|}{|D|} H(D_v)$$

选择信息增益最大的特征进行分裂

**基尼系数**：
$$Gini(D) = 1 - \sum p(x)^2$$

#### 4.3 集成方法

| 方法 | 描述 |
|------|------|
| **Bagging** | 并行训练多个模型，投票/平均 |
| **随机森林** | Bagging + 随机特征选择 |
| **Boosting** | 串行训练，每步修正前一步错误 |
| **XGBoost** | 高效梯度提升算法 |
| **AdaBoost** | 自适应提升，调整样本权重 |

## 课程三：无监督学习、推荐系统与强化学习

### Week 1：聚类算法

#### 1.1 K-means 算法

**步骤**：
1. 随机初始化 K 个聚类中心
2. 将每个样本分配到最近的中心
3. 重新计算各簇的中心位置
4. 重复 2-3 直到收敛

**优化目标**：
$$J = \sum_{i=1}^{m} \|x^{(i)} - \mu_{c^{(i)}}\|^2$$

#### 1.2 K 值选择

- **肘部法则**：绘制 J-K 曲线，选择拐点
- **业务需求**：根据实际应用场景确定

---

### Week 2：异常检测

#### 2.1 高斯分布

$$p(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

#### 2.2 异常检测算法

**步骤**：
1. 选择特征
2. 计算各特征的 μ 和 σ²
3. 计算概率 p(x)
4. 若 p(x) < ε → 异常

---

### Week 3：推荐系统

#### 3.1 协同过滤

**核心思想**：利用用户群体的行为模式

**基于用户**：找相似用户，推荐他们喜欢的物品
**基于物品**：找相似物品，推荐给喜欢相关物品的用户

#### 3.2 矩阵分解

将用户-物品评分矩阵分解为用户矩阵和物品矩阵：
$$R \approx U \times V^T$$

### Week 4：强化学习

#### 4.1 基本概念

| 术语 | 定义 |
|------|------|
| **状态 (State)** | 环境的当前情况 |
| **动作 (Action)** | 智能体可执行的操作 |
| **奖励 (Reward)** | 执行动作后获得的反馈 |
| **策略 (Policy)** | 从状态到动作的映射 |

#### 4.2 马尔可夫决策过程 (MDP)

**Bellman 方程**：
$$V(s) = \max_a [R(s,a) + \gamma \sum_{s'} P(s'|s,a) V(s')]$$

其中 γ 为折扣因子 (discount factor)

#### 4.3 Q 学习

**Q 值函数**：在状态 s 执行动作 a 的预期累积奖励

**更新规则**：
$$Q(s,a) \leftarrow Q(s,a) + \alpha [r + \gamma \max_{a'} Q(s',a') - Q(s,a)]$$

#### 4.4 深度 Q 网络 (DQN)

用神经网络近似 Q 函数：
- 输入：状态 s
- 输出：各动作的 Q 值
